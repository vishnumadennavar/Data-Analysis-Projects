Project Name: Diwali Sales Data Analysis

Tools Used:
Python, Pandas, NumPy, Matplotlib, Seaborn

Description:
In this project I performed Exploratory Data Analysis (EDA) on Diwali sales data using Python. I cleaned the dataset, removed null values, and analyzed customer purchasing behavior based on gender, age group, state, occupation, and product category.

Key Insights:

* Female customers spent more than male customers.
* Customers aged 26–35 purchased the most.
* Highest sales came from Uttar Pradesh, Maharashtra, and Karnataka.
* Top product categories were Food, Clothing, and Electronics.
